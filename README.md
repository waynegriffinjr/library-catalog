# Library Catalog

A simple Python project that manages a collection of book resources.  
This catalog supports adding books, searching by author or title, checking availability, and generating a summary of the collection.

## Features
- Add books to the catalog
- Search by author
- Search by title (keyword matching)
- List all available books
- Summary of total resources

## How to Use
1. Create `Book` or `EBook` objects  
2. Add them to the `Catalog`  
3. Call the search and availability methods as needed

## Requirements
- Python 3.x
